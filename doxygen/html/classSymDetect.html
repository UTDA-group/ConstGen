◆ hiSymDetect()

void SymDetect::hiSymDetect ( std::vector< std::vector< MosPair >> &  symGroup) const

Hierarchy symmetry detection.

Output would contain 2 levels of hierarchy. symGroup is a vector of std::vector<MosPair> oneGroup. Where oneGroup is a group of MosPair in the same symmetry group. Each MosPair should follow a MosPattern, or it should be of self symmetry. This funtion has been also updated to contain basic passive pair symmetry.
